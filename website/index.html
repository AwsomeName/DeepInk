使用指南

三步开始使用 DeepInk，开启你的 AI 工作新时代。

1

下载并安装 DeepInk

前往下方下载区获取 Mac 安装包，拖入 Applications 文件夹即可完成安装。首次打开可能需要在「系统设置 → 隐私与安全性」中允许运行。

2

登录并配置 AI 后端

使用微信扫码或手机验证码登录。进入设置页面，选择你喜欢的 AI 后端 — 可以使用内置服务，也可以填入自己的 API Key。

3

开始使用

打开右侧 Agent 面板，告诉 AI 你想做什么。它会帮你写文档、操作浏览器、整理数据 — 一切操作都在你的许可和监视下进行。

// 示例：让 AI 帮你操作浏览器
你：帮我在百度搜索 DeepInk
AI：好的，正在操作浏览器 → 导航到百度 → 填写搜索词 → 点击搜索 ✅
你：把搜索结果整理成文档
AI：已创建「DeepInk 搜索结果.docx」📄
